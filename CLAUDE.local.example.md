# CLAUDE.local.md Template

Copy this file to `CLAUDE.local.md` for personal instructions. The copied file
is gitignored.

Use this for preferences that should not become project policy, for example:

- Preferred terminal shortcuts.
- Temporary local paths.
- Personal wording preferences.
- Local service ports that differ from the shared defaults.
- Private API-key handling notes that do not include the key itself.

Do not put project-wide rules here. Shared rules belong in:

- `AGENTS.md`
- `CLAUDE.md`
- `.claude/rules/`
- `specs/`
