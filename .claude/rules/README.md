# Scoped Rules

Use this directory for path/domain-specific rules so `CLAUDE.md` and
`AGENTS.md` stay readable.

Current rule files:

- `alpha2-flow.md` - Narrative lane and Audio lane product boundaries.
- `frontend.md` - React/UI expectations.
- `sidecar.md` - Python sidecar boundary reminders.

If a rule only applies to a narrow package, prefer a scoped file here instead
of growing the root memory files.
